\subsection{Detailed Description}
Class used for downloading a file from a server on the background thread, which may be configured to send status events to a Handler object at various stages during the download. Alternatively, it may be run as a \char`\"{}fire and forget\char`\"{} task (i.\-e. no status events are returned) \begin{DoxyAuthor}{Author}
